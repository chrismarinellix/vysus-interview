# Vysus Group Technical Challenge

Interactive technical assessment for Power Systems Engineering candidates.

## Features

- **Email-based Access Control**: Candidates enter their email to start a 60-minute timed session
- **Comprehensive Quiz**: 34 technical questions covering NER compliance, dynamic simulation, leadership, and more
- **Gamified Experience**: Points system, progress tracking, and encouragement messages
- **Automatic Submission**: Results automatically sent via Netlify Forms to recruitment team
- **Session Management**: Timer countdown with warnings at 10 and 5 minutes remaining

## Deployment

This site is designed to be deployed on Netlify with form handling:

1. Connect repository to Netlify
2. Configure form notifications in Netlify dashboard:
   - Go to Site settings > Forms > Form notifications
   - Add email notification to: chris.marinelli@vysusgroup.com
3. Deploy!

## Usage

Share the deployed URL with candidates. They will:
1. Enter their email address
2. Have 60 minutes to complete the assessment
3. Results are automatically submitted to the recruitment team
